# Стратегия развития и План будущих улучшений (ROADMAP)

Проект **Minecraft Monitoring** достиг высочайшей степени готовности базового функционала уровня Production. 
Уже полностью реализованы: асинхронный пинг (через ReactPHP), бизнес-логика рейтингов, RCON, JWT токены, сброс паролей через SMTP, **Система Отзывов**, гибкие визуальные темы.

Ниже представлен план последующей массивной экспансии, нацеленной на масштабирование аудитории и Enterprise-уровень (Phases 8 & 9):

---

## 🚀 Phase 8: Gamification Engine & Programmatic SEO

### 1. Универсальный Движок Достижений (Gamification)
Создание подлинной внутренней экономики, управляемой администратором.
- **Управление из Базы Данных**: Никакого хардкода. Полный UI для администраторов по созданию/редактированию достижений, настройке картинок и выдаваемых баллов.
- **20-30 Стартовых Ачивок**: С фокусом на престиж ("Top 1% Voter", "Legendary Voter", "Early Supporter 2026"). Очки за качественные ежедневные заходы, комментарии и правдивые голоса.
- **Защита от инфляции**: Аудит-логи начислений, защита от накрутки (анти-смурф), строгий баланс выдачи баллов.
- **Передача прав на сервер**: Ручной инструмент в Админ-панели для переноса владения сервером на другого пользователя (по запросу в поддержку).

### 2. Масштабное Программное SEO и Настраиваемый Блог
Генерация сотен целевых посадочных страниц для поисковиков, чтобы привлекать низкочастотный трафик.
- **Программные Фильтры**: Страницы вида "Minecraft 1.21 PvP servers" с уникальными Title, Description, H1 и SEO-текстами для каждой из ~1000 валидных комбинаций (версия + режим).
- **SEO Блог**: Коммерческие/информационные статьи (1500+ слов) с микроразметкой, оглавлениями и ручным управлением тегами (Title, canonical, H1) прямо из админки.

## 🔒 Phase 9: Enterprise API & Strict Auth

### 3. Защищенное Публичное API v1
Подготовка сервиса к интеграции со сторонними лаунчерами и крупными проектами.
- **API Ключи**: Выдача индивидуальных API Keys пользователям с разным уровнем доступа (Rate Limits: 60/min, 1000/min).
- **Listing & Pagination**: Производительный эндпоинт `GET /api/v1/servers` для получения отфильтрованного списка серверов без перегрузки MySQL.

### 4. Двухфакторная Аутентификация (2FA) & Backup Codes
Банковская защита для владельцев крупных серверов с балансами.
- **Google Authenticator (TOTP)**: Интеграция 2FA при входе.
- **Резервные коды (Backup Codes)**: Генерация 8 одноразовых recovery-кодов на случай утери доступа.
- **Brute-Force Protection**: Блокировка попыток подбора OTP-пароля.
- **Force 2FA**: Возможность принудительного включения 2FA для профилей, оперирующих балансами или находящихся в Топе.
